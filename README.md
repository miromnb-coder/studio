# AI LIFE OPERATOR // ULTRA GLASS

Tactical financial intelligence unit powered by Agent Engine V5.6.

## Core Intelligence
- **Autonomous Reasoning (Agent V5.6)**: Multi-intent reasoning core powered by Groq (Llama 3.3).
- **The Forge**: Dynamic runtime tool creation system.
- **Financial Forensics**: Structural analysis of receipts, statements, and billing signals.
- **Glass Intelligence UI**: High-performance translucent interface built with Next.js and Tailwind.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Intelligence**: Groq SDK + Llama 3.3 / 3.2 Vision
- **Database**: Firebase Firestore
- **Auth**: Firebase Authentication (Google + Email)
- **Styling**: Tailwind CSS + ShadCN UI + Framer Motion

## Configuration
Requires `GROQ_API_KEY` in environment variables.
Firebase configuration is managed via `src/firebase/config.ts`.

## Usage
1. **Initialize Audit**: Paste statement logs or upload screenshots in the Intelligence Hub.
2. **Execute Protocols**: Use generated transmission scripts to negotiate better rates.
3. **Automate**: Connect Gmail signals for passive ingestion.
